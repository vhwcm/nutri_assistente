# nutri_assistente

Esse é um site criado no laravel, que foi feito para auxiliar nutricionistas. Eu pedi ajuda para uma nutricionista para saber as necessidades que ela tinha e que um programa poderia auxiliar. Por isso o no site existe ferramentas para calculos rápidos de: GET, TMB, IMC e a condição atual de uma pessoa. Além disso, no site você pode salvar e depois baixar automaticamente a anamnese em pdf de todos os pacientes se você for uma nutricionista e pedir para que seja analisado por uma IA a fim de ter um resumo da anamnese e recomendações, mas claro que não se pode confiar cegamente na AI, por isso é recomendado ter o auxilo de uma nutricionista. No site é possivel também acompanhar a evolução de peso de uma pessoa a partir de um gráfico gerado automaticamente.
